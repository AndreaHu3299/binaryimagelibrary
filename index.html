Welcome to the Binary Image Library

This is a library for manipulating binary (i.e. bi-valued) images. Anything that is to do with binary images belongs here. The code is pure portable ANSI C, with no dependencies other than the standard library. The idea is it should run pretty much anywhere with no problems.

A binary image is the simplest sort of image you can create, but they still throw up interesting programming problems. Whilst some binary images, like faxes, are designed for direct human viewing, generally you will only display a binary image directly on low-end hardware. However many problems in image recognition involve binary images as an intermediate step. Binary images also crop up in path finding problems, curve fitting for vectorisation, and as a step in a basic font rendering.
