Part of the binary image processing library.

\begin{DoxyNote}{Note}
conventional image rotation by the matrix method causes destination pixels to be sub-\/samples of source pixels. This isn\textquotesingle{}t a problem with continous tone images where the new pixel values can be obtained by interpolation. However with binary or colour-\/index images, interpolation isn\textquotesingle{}t possible. The shearing method preserves the pixels, at some cost in rotational accuracy. 
\end{DoxyNote}
